## Instalación

#### Prerequisitos:

- Node.

Sigue los pasos para tener el proyecto corriendo localmente:

1. Clona el repositorio:

```
git clone https://github.com/angel-luis-dev/portafolio.git
```

2. Instala las dependencias:

```
npm install
```

3. Ejecuta el proyecto:

```
npm run dev
```
